# Snake Unity Game

A 2D arcade-style snake game built in Unity where the player collects coins, grow longer, and avoids hazards.

## Features
- Snake grows as it collects coins
- Rare coins worth extra points
- Moving hazard that chases the player
- Game over on collision with walls or hazards
- Score tracking system

## Tech Stack
- Unity (2D)
- C#
- TextMeshPro

## How to Run
1. Clone the repo
2. Open in Unity Hub
3. Load the scene from `Assets/Scenes/Example 3`
4. Press Play

## Future Improvements
